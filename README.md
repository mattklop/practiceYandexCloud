# Финальный проект: Автоматизация инфраструктуры в Yandex Cloud

Данный проект демонстрирует навыки работы с методологией IaC (Infrastructure as Code) и инструментами автоматизации конфигураций.

## Что было сделано:
1. **Terraform**: Развернуты 3 виртуальные машины (Ubuntu) в Yandex Cloud.
2. **Ansible**: Созданы и применены три роли:
    * `base`: Установка системных утилит (net-tools, curl, mc и др.) и настройка имен хостов.
    * `web`: Настройка двух серверов Nginx, отдающих персонализированную страницу "Hello from <hostname>".
    * `proxy`: Настройка балансировщика Nginx, который принимает запросы на **порту 3000** и распределяет их между веб-серверами.

## Как запустить:
1. Настройте подключение к облаку в `Terraform/main.tf`.
2. Инициализируйте и примените инфраструктуру: `terraform apply`.
3. Добавьте внешние IP адреса в `Ansible/hosts.ini`.
4. Запустите настройку конфигураций: `ansible-playbook site.yml`.

## Проверка:
Результат балансировки доступен по адресу: `http://<IP_Proxy>:3000`
